# TC-006 - Verify service details are displayed correctly

## Test Case Information

| Field         | Details                                                                            |
| ------------- | ---------------------------------------------------------------------------------- |
| Test Case ID  | TC-006                                                                             |
| Module        | Service Listings                                                                   |
| Priority      | High                                                                               |
| Test Type     | Functional                                                                         |
| Preconditions | User has access to the application and at least one gardening service is available |

---

## Test Steps

| Step | Action                                                     | Expected Result                            |
| ---- | ---------------------------------------------------------- | ------------------------------------------ |
| 1    | Navigate to the Services page                              | Services page is displayed                 |
| 2    | Select a service from the available listings               | Service details page is opened             |
| 3    | Review the service information                             | Service details are displayed              |
| 4    | Verify the service title                                   | Correct service title is displayed         |
| 5    | Verify the service description                             | Description is displayed correctly         |
| 6    | Verify any available pricing information                   | Pricing information is displayed correctly |
| 7    | Verify any available images                                | Images are displayed properly              |
| 8    | Compare the displayed information with the service listing | Information is consistent and accurate     |

---

## Expected Result

The selected service should display complete and accurate information, including service title, description, pricing (if applicable), images, and other relevant details.
