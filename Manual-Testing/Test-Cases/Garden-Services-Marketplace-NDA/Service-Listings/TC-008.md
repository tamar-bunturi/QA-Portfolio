# TC-008 - Verify user can create a service booking

## Test Case Information

| Field         | Details                                                                                         |
| ------------- | ----------------------------------------------------------------------------------------------- |
| Test Case ID  | TC-008                                                                                          |
| Module        | Booking System                                                                                  |
| Priority      | High                                                                                            |
| Test Type     | Functional                                                                                      |
| Preconditions | User is logged into the application and at least one gardening service is available for booking |

---

## Test Steps

| Step | Action                                     | Expected Result                     |
| ---- | ------------------------------------------ | ----------------------------------- |
| 1    | Navigate to the Services page              | Services page is displayed          |
| 2    | Select an available gardening service      | Service details page is opened      |
| 3    | Click the Book Service button              | Booking form is displayed           |
| 4    | Select a preferred service date and time   | Selected date and time are accepted |
| 5    | Complete all required booking fields       | Form accepts entered information    |
| 6    | Submit the booking request                 | Booking request is processed        |
| 7    | Wait for confirmation                      | Booking confirmation is displayed   |
| 8    | Navigate to My Bookings or Booking History | Newly created booking is visible    |

---

## Expected Result

The user should be able to create a service booking successfully. The booking should be saved in the system and displayed in the user's booking history with the correct service details.
