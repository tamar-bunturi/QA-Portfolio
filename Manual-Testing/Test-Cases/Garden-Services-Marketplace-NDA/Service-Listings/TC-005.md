# TC-005 - Verify user can view available gardening services

## Test Case Information

| Field         | Details                                                                            |
| ------------- | ---------------------------------------------------------------------------------- |
| Test Case ID  | TC-005                                                                             |
| Module        | Service Listings                                                                   |
| Priority      | High                                                                               |
| Test Type     | Functional                                                                         |
| Preconditions | User has access to the application and at least one gardening service is available |

---

## Test Steps

| Step | Action                                | Expected Result                                                     |
| ---- | ------------------------------------- | ------------------------------------------------------------------- |
| 1    | Navigate to the Services page         | Services page is displayed                                          |
| 2    | Wait for the service listings to load | Available services are displayed                                    |
| 3    | Review the list of services           | Service entries are visible                                         |
| 4    | Verify service information            | Service name, description, and other relevant details are displayed |
| 5    | Scroll through the service listings   | Additional services are displayed correctly                         |
| 6    | Select a service from the list        | Service details page is opened                                      |

---

## Expected Result

The user should be able to view all available gardening services. Service information should be displayed correctly and users should be able to access detailed information for each service.
