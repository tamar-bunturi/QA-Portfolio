# TC-007 - Verify user can search for gardening services

## Test Case Information

| Field         | Details                                                                          |
| ------------- | -------------------------------------------------------------------------------- |
| Test Case ID  | TC-007                                                                           |
| Module        | Service Listings                                                                 |
| Priority      | Medium                                                                           |
| Test Type     | Functional                                                                       |
| Preconditions | User has access to the application and multiple gardening services are available |

---

## Test Steps

| Step | Action                                              | Expected Result                             |
| ---- | --------------------------------------------------- | ------------------------------------------- |
| 1    | Navigate to the Services page                       | Services page is displayed                  |
| 2    | Locate the search field                             | Search field is visible and accessible      |
| 3    | Enter a valid service keyword (e.g., "Lawn Mowing") | Search query is accepted                    |
| 4    | Execute the search                                  | Search request is processed                 |
| 5    | Review the search results                           | Relevant services are displayed             |
| 6    | Compare the results with the entered keyword        | Returned services match the search criteria |
| 7    | Clear the search field                              | Full service list is displayed again        |

---

## Expected Result

The user should be able to search for gardening services using keywords. The system should return relevant services that match the entered search criteria.
