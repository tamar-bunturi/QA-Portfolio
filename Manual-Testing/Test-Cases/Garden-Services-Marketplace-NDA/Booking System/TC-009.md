# TC-009 - Verify booking details are saved correctly

## Test Case Information

| Field         | Details                                                                    |
| ------------- | -------------------------------------------------------------------------- |
| Test Case ID  | TC-009                                                                     |
| Module        | Booking System                                                             |
| Priority      | High                                                                       |
| Test Type     | Functional                                                                 |
| Preconditions | User is logged into the application and has successfully created a booking |

---

## Test Steps

| Step | Action                                     | Expected Result                                |
| ---- | ------------------------------------------ | ---------------------------------------------- |
| 1    | Create a new service booking               | Booking is created successfully                |
| 2    | Navigate to My Bookings or Booking History | Booking list is displayed                      |
| 3    | Locate the newly created booking           | Booking is visible in the list                 |
| 4    | Open the booking details                   | Booking details page is displayed              |
| 5    | Verify service information                 | Selected service is displayed correctly        |
| 6    | Verify booking date and time               | Saved date and time match the submitted values |
| 7    | Verify customer information                | Customer details are displayed correctly       |

---

## Expected Result

All booking information should be saved correctly and displayed consistently throughout the application.
