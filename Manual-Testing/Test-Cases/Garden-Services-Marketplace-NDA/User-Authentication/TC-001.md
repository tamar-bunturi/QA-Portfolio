# TC-001 - Verify user can register a new account

## Test Case Information

| Field         | Details                                                                         |
| ------------- | ------------------------------------------------------------------------------- |
| Test Case ID  | TC-001                                                                          |
| Module        | User Authentication                                                             |
| Priority      | High                                                                            |
| Test Type     | Functional                                                                      |
| Preconditions | User is not logged into the application and has access to the registration page |

---

## Test Steps

| Step | Action                            | Expected Result                                                      |
| ---- | --------------------------------- | -------------------------------------------------------------------- |
| 1    | Navigate to the registration page | Registration form is displayed                                       |
| 2    | Enter a valid email address       | Email is accepted                                                    |
| 3    | Enter a valid password            | Password is accepted                                                 |
| 4    | Complete all required fields      | Form accepts entered information                                     |
| 5    | Click Register                    | Registration request is processed                                    |
| 6    | Observe the system response       | Account is created successfully                                      |
| 7    | Verify user access                | User is redirected to the appropriate page or logged in successfully |

---

## Expected Result

The user should be able to register a new account successfully using valid information. A new account should be created and accessible for future login.
