# TC-004 - Verify user can log out successfully

## Test Case Information

| Field         | Details                             |
| ------------- | ----------------------------------- |
| Test Case ID  | TC-004                              |
| Module        | User Authentication                 |
| Priority      | High                                |
| Test Type     | Functional                          |
| Preconditions | User is logged into the application |

---

## Test Steps

| Step | Action                                                                         | Expected Result                                             |
| ---- | ------------------------------------------------------------------------------ | ----------------------------------------------------------- |
| 1    | Log in to the application using valid credentials                              | User is authenticated successfully                          |
| 2    | Locate the Logout option in the application                                    | Logout option is visible                                    |
| 3    | Click Logout                                                                   | Logout request is processed                                 |
| 4    | Wait for the operation to complete                                             | User session is terminated                                  |
| 5    | Observe the application response                                               | User is redirected to the login page or public landing page |
| 6    | Attempt to access a protected page using the browser Back button or direct URL | Access is denied and user is redirected to the login page   |

---

## Expected Result

The user should be logged out successfully, the active session should be terminated, and access to authenticated areas of the application should no longer be available.

## Actual Result

To be completed during execution.
