# TC-002 - Verify user can log in with valid credentials

## Test Case Information

| Field         | Details                           |
| ------------- | --------------------------------- |
| Test Case ID  | TC-002                            |
| Module        | User Authentication               |
| Priority      | High                              |
| Test Type     | Functional                        |
| Preconditions | User account exists and is active |

---

## Test Steps

| Step | Action                              | Expected Result                            |
| ---- | ----------------------------------- | ------------------------------------------ |
| 1    | Navigate to the login page          | Login form is displayed                    |
| 2    | Enter a valid email address         | Email is accepted                          |
| 3    | Enter the correct password          | Password is accepted                       |
| 4    | Click Login                         | Login request is processed                 |
| 5    | Wait for authentication to complete | User is authenticated successfully         |
| 6    | Observe the landing page            | User is redirected to the appropriate page |
| 7    | Verify account access               | User can access authenticated features     |

---

## Expected Result

The user should be able to log in successfully using valid credentials and gain access to the application's authenticated features.

---

## Actual Result

To be completed during execution.
