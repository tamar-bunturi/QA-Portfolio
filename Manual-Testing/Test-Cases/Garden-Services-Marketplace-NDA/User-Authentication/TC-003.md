# TC-003 - Verify user cannot log in with invalid credentials

## Test Case Information

| Field         | Details                   |
| ------------- | ------------------------- |
| Test Case ID  | TC-003                    |
| Module        | User Authentication       |
| Priority      | High                      |
| Test Type     | Negative Testing          |
| Preconditions | User is on the login page |

---

## Test Steps

| Step | Action                                               | Expected Result                              |
| ---- | ---------------------------------------------------- | -------------------------------------------- |
| 1    | Navigate to the login page                           | Login form is displayed                      |
| 2    | Enter an invalid email address or incorrect password | Credentials are accepted in the input fields |
| 3    | Click Login                                          | Login request is processed                   |
| 4    | Observe the system response                          | Authentication fails                         |
| 5    | Review the displayed message                         | Appropriate error message is shown           |
| 6    | Verify access permissions                            | User remains unauthenticated                 |

---

## Expected Result

The system should reject invalid credentials and display a clear error message. The user should not gain access to authenticated areas of the application.
