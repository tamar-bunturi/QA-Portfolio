# TC-004 - Verify user can delete an existing document

## Test Case Information

| Field         | Details                                                              |
| ------------- | -------------------------------------------------------------------- |
| Test Case ID  | TC-004                                                               |
| Module        | Document Management                                                  |
| Priority      | High                                                                 |
| Test Type     | Functional                                                           |
| Preconditions | User is logged into the application and at least one document exists |

---

## Test Steps

| Step | Action                                        | Expected Result                                             |
| ---- | --------------------------------------------- | ----------------------------------------------------------- |
| 1    | Navigate to the Documents or Projects section | Documents page is displayed                                 |
| 2    | Locate an existing document                   | Document is visible in the list                             |
| 3    | Select the Delete option for the document     | Delete confirmation dialog is displayed                     |
| 4    | Confirm the deletion                          | Delete request is processed                                 |
| 5    | Refresh the document list                     | Updated document list is displayed                          |
| 6    | Search for the deleted document               | Document is no longer available in the active document list |

---

## Expected Result

The user should be able to delete an existing document successfully. The deleted document should no longer appear in the active document list and should not be accessible from the document management interface.
