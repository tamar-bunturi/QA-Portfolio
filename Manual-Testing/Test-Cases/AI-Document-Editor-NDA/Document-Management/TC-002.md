# TC-002 - Verify user can save changes to an existing document

## Test Case Information

| Field         | Details                                                                               |
| ------------- | ------------------------------------------------------------------------------------- |
| Test Case ID  | TC-002                                                                                |
| Module        | Document Management                                                                   |
| Priority      | High                                                                                  |
| Test Type     | Functional                                                                            |
| Preconditions | User is logged into the application and an existing document is available for editing |

---

## Test Steps

| Step | Action                                        | Expected Result                          |
| ---- | --------------------------------------------- | ---------------------------------------- |
| 1    | Navigate to the Documents or Projects section | Documents page is displayed              |
| 2    | Open an existing document                     | Document editor is opened                |
| 3    | Modify the document title or content          | Changes are accepted                     |
| 4    | Click Save                                    | Save request is processed                |
| 5    | Refresh the page or reopen the document       | Updated document is loaded               |
| 6    | Review the document content                   | Modified changes are displayed correctly |

---

## Expected Result

The user should be able to save changes to an existing document successfully. All modifications should persist after refreshing the page or reopening the document.
