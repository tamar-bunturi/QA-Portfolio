# TC-003 - Verify user can edit an existing document

## Test Case Information

| Field         | Details                                                                   |
| ------------- | ------------------------------------------------------------------------- |
| Test Case ID  | TC-003                                                                    |
| Module        | Document Management                                                       |
| Priority      | High                                                                      |
| Test Type     | Functional                                                                |
| Preconditions | User is logged into the application and an existing document is available |

---

## Test Steps

| Step | Action                                        | Expected Result                            |
| ---- | --------------------------------------------- | ------------------------------------------ |
| 1    | Navigate to the Documents or Projects section | Documents page is displayed                |
| 2    | Open an existing document                     | Document editor is opened                  |
| 3    | Modify the document title                     | Updated title is accepted                  |
| 4    | Modify the document content                   | Updated content is accepted                |
| 5    | Verify changes are visible in the editor      | Updated information is displayed correctly |
| 6    | Click Save                                    | Save request is processed successfully     |
| 7    | Reopen the document                           | Updated title and content are displayed    |

---

## Expected Result

The user should be able to edit an existing document successfully. All modifications should be reflected in the document and remain available after saving and reopening.
