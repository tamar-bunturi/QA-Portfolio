# TC-001 - Verify user can create a new document

## Test Case Information

| Field         | Details                                                                    |
| ------------- | -------------------------------------------------------------------------- |
| Test Case ID  | TC-001                                                                     |
| Module        | Document Management                                                        |
| Priority      | High                                                                       |
| Test Type     | Functional                                                                 |
| Preconditions | User is logged into the application and has permission to create documents |

---

## Test Steps

| Step | Action                                        | Expected Result                        |
| ---- | --------------------------------------------- | -------------------------------------- |
| 1    | Navigate to the Documents or Projects section | Documents page is displayed            |
| 2    | Click the Create New Document button          | New document form or editor is opened  |
| 3    | Enter a valid document title                  | Title is accepted                      |
| 4    | Enter document content                        | Content is accepted                    |
| 5    | Click Save or Create                          | Document creation request is processed |
| 6    | Navigate back to the document list            | Document list is displayed             |
| 7    | Locate the newly created document             | New document appears in the list       |

---

## Expected Result

The user should be able to create a new document successfully. The document should be saved and displayed in the document list with the entered title and content.
