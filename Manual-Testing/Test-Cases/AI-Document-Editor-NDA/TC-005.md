# TC-005 - Verify uploaded image is visible in the Admin Panel conversation view

## Test Case Information

| Field         | Details                                                          |
| ------------- | ---------------------------------------------------------------- |
| Test Case ID  | TC-005                                                           |
| Module        | AI Chat Assistant / Admin Panel                                  |
| Priority      | High                                                             |
| Test Type     | Functional                                                       |
| Preconditions | Customer chat widget and Admin Panel are available and connected |

---

## Test Steps

| Step | Action                                                 | Expected Result                       |
| ---- | ------------------------------------------------------ | ------------------------------------- |
| 1    | Open the customer chat widget                          | Chat widget opens successfully        |
| 2    | Upload a valid image file                              | Image is uploaded successfully        |
| 3    | Send a message together with the image                 | Message is sent successfully          |
| 4    | Open the corresponding conversation in the Admin Panel | Conversation is displayed             |
| 5    | Locate the customer's message                          | Customer message is visible           |
| 6    | Review the message attachments                         | Uploaded image is displayed correctly |
| 7    | Open the image preview (if supported)                  | Image preview opens successfully      |

---

## Expected Result

Images uploaded by customers should be displayed in the Admin Panel conversation view. Support agents should be able to view the same image attachments visible to the customer.
