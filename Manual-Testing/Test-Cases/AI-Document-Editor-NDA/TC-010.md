# TC-010 - Verify deleted documents are excluded from AI Search results

## Test Case Information

| Field         | Details                                                                                    |
| ------------- | ------------------------------------------------------------------------------------------ |
| Test Case ID  | TC-010                                                                                     |
| Module        | AI Search                                                                                  |
| Priority      | High                                                                                       |
| Test Type     | Functional                                                                                 |
| Preconditions | AI Search is enabled and at least one document has been deleted and moved to the Bin/Trash |

---

## Test Steps

| Step | Action                                                     | Expected Result                                 |
| ---- | ---------------------------------------------------------- | ----------------------------------------------- |
| 1    | Navigate to the Library page                               | Library page is displayed                       |
| 2    | Create or identify an existing document                    | Document is available in the Library            |
| 3    | Delete the document                                        | Document is moved to the Bin/Trash              |
| 4    | Navigate to the AI Search page                             | AI Search interface is displayed                |
| 5    | Enter the name of the deleted document in the search field | Search query is accepted                        |
| 6    | Execute the search                                         | Search request is processed                     |
| 7    | Review the search results                                  | Results are displayed                           |
| 8    | Verify returned documents                                  | Deleted document does not appear in the results |

---

## Expected Result

Documents that have been deleted and moved to the Bin/Trash should not appear in AI Search results. Only active documents should be returned to users.
