# TC-003 - Verify admin can reply to customer through AI Chat

## Test Case Information

| Field         | Details                                                          |
| ------------- | ---------------------------------------------------------------- |
| Test Case ID  | TC-003                                                           |
| Module        | AI Chat Assistant                                                |
| Priority      | High                                                             |
| Test Type     | Functional                                                       |
| Preconditions | Customer chat widget and Admin Panel are available and connected |

---

## Test Steps

| Step | Action                                                 | Expected Result                          |
| ---- | ------------------------------------------------------ | ---------------------------------------- |
| 1    | Open the customer chat widget                          | Chat widget opens successfully           |
| 2    | Send a text message from the customer chat             | Message is sent successfully             |
| 3    | Open the corresponding conversation in the Admin Panel | Conversation is displayed                |
| 4    | Type a reply in the Admin Panel                        | Reply is entered successfully            |
| 5    | Click Send                                             | Reply is sent successfully               |
| 6    | Return to the customer chat widget                     | Customer chat remains active             |
| 7    | Observe the conversation                               | Admin reply appears in the customer chat |

---

## Expected Result

The customer should receive the admin's reply in real time, and the message should appear in the conversation history on both the customer and admin sides.
