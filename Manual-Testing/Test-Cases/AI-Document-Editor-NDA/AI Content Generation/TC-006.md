# TC-006 - Verify AI-generated document content matches the user-provided topic

## Test Case Information

| Field         | Details                                                                          |
| ------------- | -------------------------------------------------------------------------------- |
| Test Case ID  | TC-006                                                                           |
| Module        | AI Content Generation                                                            |
| Priority      | High                                                                             |
| Test Type     | Functional                                                                       |
| Preconditions | User is logged into the application and AI content generation feature is enabled |

---

## Test Steps

| Step | Action                                                                                    | Expected Result                                 |
| ---- | ----------------------------------------------------------------------------------------- | ----------------------------------------------- |
| 1    | Navigate to the Documents or Projects section                                             | Documents page is displayed                     |
| 2    | Create a new document or open an existing document                                        | Document editor is opened                       |
| 3    | Open the AI content generation feature                                                    | AI prompt interface is displayed                |
| 4    | Enter a specific topic (e.g., "Generate content about software testing and QA practices") | Prompt is accepted                              |
| 5    | Click Generate                                                                            | AI generation request is processed              |
| 6    | Wait for content generation to complete                                                   | Generated content is displayed                  |
| 7    | Review the generated content                                                              | Content is relevant to the requested topic      |
| 8    | Compare the generated content with the original prompt                                    | Main topic and context are reflected accurately |

---

## Expected Result

The AI should generate content that aligns with the topic provided by the user. The generated document should be relevant, coherent, and focused on the requested subject.
