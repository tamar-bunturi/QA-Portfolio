# TC-004 - Verify customer can upload an image in AI Chat

## Test Case Information

| Field         | Details                                                                     |
| ------------- | --------------------------------------------------------------------------- |
| Test Case ID  | TC-004                                                                      |
| Module        | AI Chat Assistant                                                           |
| Priority      | High                                                                        |
| Test Type     | Functional                                                                  |
| Preconditions | Customer chat widget is available and image upload functionality is enabled |

---

## Test Steps

| Step | Action                                     | Expected Result                    |
| ---- | ------------------------------------------ | ---------------------------------- |
| 1    | Open the customer chat widget              | Chat widget opens successfully     |
| 2    | Click the attachment/upload button         | File selection dialog is displayed |
| 3    | Select a valid image file (JPG, PNG, JPEG) | File is accepted by the system     |
| 4    | Submit the image                           | Upload process starts successfully |
| 5    | Wait for the upload to complete            | Upload completes without errors    |
| 6    | Observe the chat conversation              | Uploaded image appears in the chat |
| 7    | Verify image preview                       | Image is displayed correctly       |

---

## Expected Result

The customer should be able to upload a supported image file successfully. The uploaded image should appear in the conversation and be visible within the chat interface.
