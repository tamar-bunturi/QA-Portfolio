# TC-012 - Verify deleted documents are excluded from search results

## Test Case Information

| Field         | Details                                                                             |
| ------------- | ----------------------------------------------------------------------------------- |
| Test Case ID  | TC-012                                                                              |
| Module        | Search and Library                                                                  |
| Priority      | High                                                                                |
| Test Type     | Functional                                                                          |
| Preconditions | User is logged into the application and at least one document exists in the library |

---

## Test Steps

| Step | Action                                               | Expected Result                                    |
| ---- | ---------------------------------------------------- | -------------------------------------------------- |
| 1    | Navigate to the Library page                         | Library page is displayed                          |
| 2    | Locate an existing document                          | Document is visible in the library                 |
| 3    | Delete the document                                  | Document is moved to the Bin/Trash                 |
| 4    | Navigate back to the search page                     | Search page is displayed                           |
| 5    | Enter the deleted document title in the search field | Search query is accepted                           |
| 6    | Execute the search                                   | Search request is processed                        |
| 7    | Review the search results                            | Results are displayed                              |
| 8    | Verify the deleted document is not listed            | Deleted document does not appear in search results |

---

## Expected Result

Documents that have been deleted and moved to the Bin/Trash should not be returned in search results. Only active documents should be searchable and visible to users.
