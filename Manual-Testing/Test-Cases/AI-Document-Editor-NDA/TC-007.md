# TC-007 - Verify AI Assistant returns accurate product details for an existing Shopify product

## Test Case Information

| Field         | Details                                                                                                                |
| ------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Test Case ID  | TC-007                                                                                                                 |
| Module        | Shopify Integration / AI Assistant                                                                                     |
| Priority      | High                                                                                                                   |
| Test Type     | Functional                                                                                                             |
| Preconditions | AI Assistant is enabled and connected to the Shopify product catalog. At least one active product exists in the store. |

---

## Test Steps

| Step | Action                                                  | Expected Result                                               |
| ---- | ------------------------------------------------------- | ------------------------------------------------------------- |
| 1    | Open the customer-facing AI Assistant chat              | AI Assistant is displayed successfully                        |
| 2    | Enter the name of an existing Shopify product           | Query is accepted                                             |
| 3    | Submit the query                                        | AI Assistant processes the request                            |
| 4    | Wait for the response                                   | Product information is returned                               |
| 5    | Review the product details provided by the AI Assistant | Product name, description, and relevant details are displayed |
| 6    | Compare the response with the Shopify product page      | Information matches the Shopify catalog                       |
| 7    | Verify no incorrect product information is shown        | Product details are accurate                                  |

---

## Expected Result

The AI Assistant should return accurate and up-to-date product information that matches the corresponding product in the Shopify catalog.
