# TC-006 - Verify customer can search for a product using AI Assistant

## Test Case Information

| Field         | Details                                                              |
| ------------- | -------------------------------------------------------------------- |
| Test Case ID  | TC-006                                                               |
| Module        | Shopify Integration / AI Assistant                                   |
| Priority      | High                                                                 |
| Test Type     | Functional                                                           |
| Preconditions | AI Assistant is enabled and connected to the Shopify product catalog |

---

## Test Steps

| Step | Action                                      | Expected Result                          |
| ---- | ------------------------------------------- | ---------------------------------------- |
| 1    | Open the customer-facing AI Assistant chat  | AI Assistant is displayed successfully   |
| 2    | Enter the name of an existing product       | Query is accepted                        |
| 3    | Submit the query                            | AI Assistant processes the request       |
| 4    | Wait for the response                       | Response is generated successfully       |
| 5    | Review the returned product information     | Product details are displayed            |
| 6    | Compare the result with the Shopify catalog | Returned information matches the catalog |

---

## Expected Result

The AI Assistant should successfully find products that exist in the Shopify catalog and return accurate product information to the customer.
