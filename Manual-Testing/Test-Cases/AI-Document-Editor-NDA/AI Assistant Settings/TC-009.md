# TC-009 - Verify user can change the default AI model

## Test Case Information

| Field         | Details                                                                     |
| ------------- | --------------------------------------------------------------------------- |
| Test Case ID  | TC-009                                                                      |
| Module        | AI Assistant Settings                                                       |
| Priority      | High                                                                        |
| Test Type     | Functional                                                                  |
| Preconditions | User is logged into the application and has access to AI Assistant Settings |

---

## Test Steps

| Step | Action                                           | Expected Result                                               |
| ---- | ------------------------------------------------ | ------------------------------------------------------------- |
| 1    | Navigate to Settings                             | Settings page is displayed                                    |
| 2    | Open AI Assistant Settings                       | AI configuration options are displayed                        |
| 3    | Locate the default AI model setting              | Current model is displayed                                    |
| 4    | Select a different available AI model            | New model is selected successfully                            |
| 5    | Click Save                                       | Configuration changes are saved                               |
| 6    | Refresh the page or reopen AI Assistant Settings | Settings page reloads successfully                            |
| 7    | Review the selected AI model                     | Newly selected model is displayed as the active default model |

---

## Expected Result

The user should be able to change the default AI model successfully. The selected model should be saved and displayed as the active default model after saving and reloading the settings page.
