# TC-011 - Verify search returns matching documents based on document title

## Test Case Information

| Field         | Details                                                                             |
| ------------- | ----------------------------------------------------------------------------------- |
| Test Case ID  | TC-011                                                                              |
| Module        | Search and Library                                                                  |
| Priority      | High                                                                                |
| Test Type     | Functional                                                                          |
| Preconditions | User is logged into the application and at least one document exists in the library |

---

## Test Steps

| Step | Action                                               | Expected Result                    |
| ---- | ---------------------------------------------------- | ---------------------------------- |
| 1    | Navigate to the Library page                         | Library page is displayed          |
| 2    | Locate a document with a known title                 | Document is visible in the library |
| 3    | Enter the document title in the search field         | Search query is accepted           |
| 4    | Execute the search                                   | Search request is processed        |
| 5    | Review the search results                            | Matching documents are displayed   |
| 6    | Compare the results with the original document title | Correct document is returned       |

---

## Expected Result

The search feature should return documents that match the entered title. The relevant document should be displayed in the search results and be accessible to the user.
