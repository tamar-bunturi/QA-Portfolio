# TC-010 - Verify selected AI model persists after saving and page refresh

## Test Case Information

| Field         | Details                                                                     |
| ------------- | --------------------------------------------------------------------------- |
| Test Case ID  | TC-010                                                                      |
| Module        | AI Assistant Settings                                                       |
| Priority      | High                                                                        |
| Test Type     | Functional                                                                  |
| Preconditions | User is logged into the application and has access to AI Assistant Settings |

---

## Test Steps

| Step | Action                                                 | Expected Result                            |
| ---- | ------------------------------------------------------ | ------------------------------------------ |
| 1    | Navigate to Settings                                   | Settings page is displayed                 |
| 2    | Open AI Assistant Settings                             | AI configuration options are displayed     |
| 3    | Select a different AI model from the available options | New model is selected successfully         |
| 4    | Click Save                                             | Configuration changes are saved            |
| 5    | Refresh the browser page                               | Settings page reloads successfully         |
| 6    | Return to AI Assistant Settings                        | Settings are displayed                     |
| 7    | Review the active AI model                             | Previously selected model remains selected |
| 8    | Open a new AI Assistant session                        | Session uses the selected AI model         |

---

## Expected Result

The selected AI model should persist after saving, refreshing the page, reopening settings, and starting a new AI Assistant session.
