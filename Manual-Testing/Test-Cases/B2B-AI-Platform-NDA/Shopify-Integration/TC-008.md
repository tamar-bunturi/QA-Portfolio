# TC-008 - Verify AI Assistant handles unavailable or non-existent products correctly

## Test Case Information

| Field         | Details                                                              |
| ------------- | -------------------------------------------------------------------- |
| Test Case ID  | TC-008                                                               |
| Module        | Shopify Integration / AI Assistant                                   |
| Priority      | High                                                                 |
| Test Type     | Functional                                                           |
| Preconditions | AI Assistant is enabled and connected to the Shopify product catalog |

---

## Test Steps

| Step | Action                                                          | Expected Result                                                        |
| ---- | --------------------------------------------------------------- | ---------------------------------------------------------------------- |
| 1    | Open the customer-facing AI Assistant chat                      | AI Assistant is displayed successfully                                 |
| 2    | Enter a product name that does not exist in the Shopify catalog | Query is accepted                                                      |
| 3    | Submit the query                                                | AI Assistant processes the request                                     |
| 4    | Wait for the response                                           | Response is generated successfully                                     |
| 5    | Review the AI Assistant's response                              | Product is reported as unavailable or not found                        |
| 6    | Verify the response content                                     | Response is clear, accurate, and does not reference unrelated products |

---

## Expected Result

The AI Assistant should clearly inform the customer that the requested product is unavailable or does not exist in the Shopify catalog. No incorrect product recommendations or misleading information should be displayed.
