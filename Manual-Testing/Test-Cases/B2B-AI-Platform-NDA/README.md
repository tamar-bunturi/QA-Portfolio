
# B2B AI Platform (NDA)

## Project Overview

A B2B web platform integrating AI-powered capabilities including:

* AI Chat Assistant
* AI Search Engine
* Shopify Integration
* Customer Support Workflows

The platform enables users to discover content, search products, and interact with AI-powered conversational assistants.
