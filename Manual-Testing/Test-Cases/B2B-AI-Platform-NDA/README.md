
# B2B AI Platform (NDA)

## Project Overview

A B2B web platform integrating AI-powered capabilities including:

* AI Chat Assistant
* AI Search Engine
* Shopify Integration
* Customer Support Workflows

The platform enables users to discover content, search products, and interact with AI-powered conversational assistants.

> Company and client information have been omitted due to NDA restrictions.

## Modules Covered

### AI Chat Assistant

* TC-001 - Verify customer can send text message
* TC-002 - Verify admin receives customer message
* TC-003 - Verify admin can reply to customer
* TC-004 - Verify customer can upload an image
* TC-005 - Verify uploaded image is visible in Admin Panel

### Shopify Integration

* TC-006 - Verify customer can search for a product using AI Assistant
* TC-007 - Verify AI Assistant returns accurate product details
* TC-008 - Verify AI Assistant handles unavailable products correctly

### AI Search

* TC-009 - Verify AI Search returns relevant results for an exact keyword match
* TC-010 - Verify deleted documents are excluded from AI Search results

## Test Design Techniques Used

* Functional Testing
* Positive Testing
* Negative Testing
* End-to-End Testing
* Exploratory Testing
* AI Response Validation
* Search Relevance Testing
* Integration Testing
