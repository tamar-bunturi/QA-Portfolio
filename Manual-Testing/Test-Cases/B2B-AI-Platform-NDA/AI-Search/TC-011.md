# TC-011 - Verify AI Search returns relevant results for a partial keyword match

## Test Case Information

| Field         | Details                                                         |
| ------------- | --------------------------------------------------------------- |
| Test Case ID  | TC-011                                                          |
| Module        | AI Search                                                       |
| Priority      | Medium                                                          |
| Test Type     | Functional                                                      |
| Preconditions | AI Search is enabled and the library contains indexed documents |

---

## Test Steps

| Step | Action                                                                                   | Expected Result                                       |
| ---- | ---------------------------------------------------------------------------------------- | ----------------------------------------------------- |
| 1    | Navigate to the AI Search page                                                           | AI Search interface is displayed                      |
| 2    | Enter a partial keyword from an existing document title (e.g., "IST" instead of "ISTQB") | Search query is accepted                              |
| 3    | Execute the search                                                                       | Search request is processed                           |
| 4    | Review the search results                                                                | Relevant documents are displayed                      |
| 5    | Compare the results with the available library documents                                 | Documents containing the partial keyword are returned |

---

## Expected Result

The AI Search feature should return relevant documents when a user enters a partial keyword. Search results should include documents containing the matching text and should be ranked by relevance.
