# TC-009 - Verify AI Search returns relevant results for an exact keyword match

## Test Case Information

| Field         | Details                                                                    |
| ------------- | -------------------------------------------------------------------------- |
| Test Case ID  | TC-009                                                                     |
| Module        | AI Search                                                                  |
| Priority      | High                                                                       |
| Test Type     | Functional                                                                 |
| Preconditions | AI Search is enabled and the library contains documents indexed for search |

---

## Test Steps

| Step | Action                                                                 | Expected Result                                     |
| ---- | ---------------------------------------------------------------------- | --------------------------------------------------- |
| 1    | Navigate to the AI Search page                                         | AI Search interface is displayed                    |
| 2    | Enter an exact keyword that exists in a document title (e.g., "ISTQB") | Search query is accepted                            |
| 3    | Execute the search                                                     | Search request is processed                         |
| 4    | Review the search results                                              | Relevant documents are displayed                    |
| 5    | Compare results with available library documents                       | Documents containing the exact keyword are returned |
| 6    | Verify the ranking of results                                          | Most relevant documents appear first                |

---

## Expected Result

The AI Search feature should return documents containing the exact keyword entered by the user. Results should be relevant and ranked according to their relevance to the search term.
