### App Overview
A mobile social networking application focused on travel and 
location-based content sharing. The platform allows users to 
share posts and stories, discover places via an interactive map, 
connect with other travelers through a real-time messenger, 
and manage their travel profile. The app includes WebSocket-based 
real-time communication and AI-assisted features.

### Platform
- Mobile Application (Android)

### Testing Focus Areas
- User Registration & Authentication (OTP, session management)
- Social Feed (posts, stories, pagination, video autoplay)
- Real-time Messenger (WebSocket, voice messages, group chats)
- Profile Management (photo upload, edit profile, follow/unfollow)
- Map & Search functionality
- Push Notifications
- Negative testing & edge cases
