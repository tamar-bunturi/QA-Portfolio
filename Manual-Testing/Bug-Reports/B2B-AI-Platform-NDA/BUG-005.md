# BUG-005 - First customer-uploaded image is not displayed in the Admin Panel conversation view

| Field               | Details                                   |
| ------------------- | ----------------------------------------- |
| **Module**          | Shopify Integration / AI Chat Attachments |
| **Category**        | Functional                                |
| **Severity**        | High                                      |
| **Priority**        | High                                      |
| **Reproducibility** | 100%                                      |

---

## Description

When a customer uploads the first image in a conversation through the Shopify-integrated AI Chat, the image is visible in the customer-facing chat but is not displayed in the Admin Panel conversation view. As a result, agents can only see the text message and cannot view the referenced product image.

---

## Preconditions

- AI Assistant is enabled
- Admin Panel is open
- Customer chat widget is active
- Shopify integration is configured and operational

---

## Steps to Reproduce

1. Open the customer-facing AI Assistant chat
2. Upload an image
3. Send a message such as "Do you have this product?"
4. Open the corresponding conversation in the Admin Panel
5. Review the customer message and attachments

---

## Expected Result

Any image uploaded by the customer should be displayed in the Admin Panel conversation view.

Admin users should have access to the same image attachments visible in the customer chat, ensuring a consistent conversation history across both interfaces.

---

## Actual Result

The customer can see the uploaded image in the chat.

However, the Admin Panel displays only the text message and does not display the uploaded image attachment. The agent cannot view the image and therefore cannot determine which product the customer is referencing.

---

## Impact

> Customer-uploaded image attachments are not visible to support agents, preventing them from viewing referenced products and negatively affecting customer support effectiveness.
