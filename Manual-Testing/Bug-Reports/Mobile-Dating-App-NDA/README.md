## App Overview

This repository contains sample bug reports identified during manual testing of a mobile dating application.

The platform is designed to help people connect, communicate, and build meaningful relationships in a secure environment. User safety is a core feature of the application, with identity verification mechanisms implemented to ensure that only verified users can access key communication and dating features.

## Scope of Testing

The testing covered the following areas:

* User Registration & Authentication
* Profile Management
* Identity Verification Flows
* Photo Upload & Management
* Messaging & Chat Functionality
* Support & Customer Service Features
* Settings & Preferences
* UI/UX Validation
* Accessibility & Usability Testing
