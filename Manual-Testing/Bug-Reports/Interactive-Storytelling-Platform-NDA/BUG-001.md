# BUG-001 - The Founder's LinkedIn Icon Does Not Open LinkedIn Profile

| Field           | Details    |
| --------------- | ---------- |
| Module          | Our Team   |
| Category        | Functional |
| Severity        | Medium     |
| Priority        | Medium     |
| Reproducibility | 100%       |

---

## Preconditions

* User has access to the website.
* User is on the About page.
* The Our Team section is available.

---

## Steps to Reproduce

1. Open the website.
2. Navigate to the **Our Team** section.
3. Locate the founder's profile card.
4. Click the LinkedIn icon displayed on the profile card.
5. Observe the behavior.

---

## Expected Result

Clicking the LinkedIn icon should open the founder's LinkedIn profile in a new browser tab.

---

## Actual Result

Clicking the LinkedIn icon does not open any LinkedIn profile and no action is performed.

---

## Impact

Users are unable to access the founder's professional profile through the provided LinkedIn icon.

This may reduce credibility, limit networking opportunities, and create the impression that the link is broken or incomplete.
