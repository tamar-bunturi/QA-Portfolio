# BUG-004 - Footer Decorative Element Becomes Visible Outside Intended Page Boundary

| Field | Details |
|---------|---------|
| Module | Footer |
| Category | UI / Layout |
| Severity | Low |
| Priority | Low |
| Reproducibility | 100% |

---

## Preconditions

- User has access to the website.
- User is on a page containing the footer section.

---

## Steps to Reproduce

1. Open the website.
2. Navigate to any page containing the footer.
3. Scroll down to the footer section.
4. Observe the animated decorative visual element near the bottom of the page.

---

## Expected Result

The decorative footer element should remain fully contained within its designated section and should not be visible outside the intended page boundaries.

Animations should remain visually clean and should not expose hidden portions of graphical assets.

---

## Actual Result

During the animation, parts of the decorative footer element become visible below the footer boundary.

As the element moves backward in the animation sequence, additional portions of the graphic appear outside the intended layout area, creating a visually inconsistent presentation.

---

## Impact

Although functionality is not affected, the issue negatively impacts the visual quality of the website and may create the impression of an unfinished or broken layout.

Users may notice unexpected graphical artifacts below the footer, which can reduce the overall polish and professionalism of the interface.
