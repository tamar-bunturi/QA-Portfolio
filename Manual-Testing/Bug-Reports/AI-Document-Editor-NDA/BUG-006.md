# BUG-006 - AI generates content that does not match the user-provided topic

| Field | Details |
|---------|---------|
| Module | AI File Generator |
| Category | Functional |
| Severity | High |
| Priority | High |
| Reproducibility | 100% |

---

## Preconditions

1. User is logged into the application.
2. User has access to Projects.
3. AI file generation feature is enabled.

---

## Steps to Reproduce

1. Navigate to the application.
2. Open **Projects**.
3. Click the **"+"** button and create a new file.
4. Enter a file title.
5. Click the **AI Assistant** icon.
6. In the topic field enter a specific request, for example:
   - "Please generate text about the tech field and QA Tester role."
7. Submit the request.
8. Wait until AI content generation is completed.
9. Click **Generate** and review the produced manuscript.

---

## Expected Result

The generated file should accurately follow the user-provided topic and contain content specifically related to the technology industry and QA Tester responsibilities.

---

## Actual Result

The generated manuscript contains unrelated, partially relevant, or incorrect information that does not fully correspond to the topic entered by the user.

---

## Impact

- Users receive inaccurate AI-generated content.
- Additional manual editing is required.
- Reduces trust in AI-generated manuscript functionality.
- Can lead to incorrect documents being created.
