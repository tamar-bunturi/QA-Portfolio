## AI-Powered Document Editor

> ⚠️ NDA Notice: This project is covered by a Non-Disclosure

### App Overview
A web-based document editing platform that integrates multiple 
AI assistants directly into the editor environment. Users can 
write and edit documents while simultaneously interacting with 
AI to request content generation, text corrections, character 
name changes, formatting adjustments, and structural improvements. 
The platform bridges traditional document editing with real-time 
AI-guided assistance.

### Testing Focus Areas
- AI response accuracy within the editor context
- Formatting and rendering consistency
- UI behavior during simultaneous editing and AI interaction
- Dropdown and toolbar functionality
- Document integrity after AI-suggested changes
